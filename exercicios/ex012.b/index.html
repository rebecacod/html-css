Oque é a linguagem html?

html é uma linguagem de marcação de texto, essa linguagem é usada juntamente com as css e o javascript geralmente andam lado a lado, mas não pode ser chamada de linguagem de programação, afinal ela não pode ser usada pra programar ela é usada especificamente para configuração de um site na web, ela cuida da parte do conteudo enquato as css cuidam do estilo deixamdo a pagina mais agradavel aos olhos do leitor sendo parceira da ux e da ui
